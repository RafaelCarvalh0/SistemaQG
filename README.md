# 📌 SistemaQG

> Sistema web para registro e controle de pontos, criado como proposta para otimizar a comunicação em uma empresa de segurança.

---

## 🧠 Contexto

Este projeto foi desenvolvido no início dos meus estudos, durante minha atuação em uma empresa de segurança.

Na época, o controle de pontos era realizado via ligações telefônicas, o que frequentemente causava congestionamento na linha devido ao grande volume de chamadas simultâneas.

Como alternativa, desenvolvi este sistema web com o objetivo de centralizar e simplificar esse processo.

> ⚠️ Este projeto foi criado como prova de conceito (POC), contendo apenas funcionalidades básicas e sem foco em segurança.

---
